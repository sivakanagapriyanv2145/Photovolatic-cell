# 🌞 Solar Panel Fault Detection (ELPV Dataset - Simple ML Project)

This is a very simple machine learning project inspired by the ELPV dataset concept. It predicts whether a solar panel is in **good condition or faulty** based on environmental parameters.

---

## 🚀 Project Idea

The model uses basic features like:
- Irradiance
- Temperature
- Voltage
- Current

It classifies the solar panel status as:
- ✅ GOOD (working properly)
- ⚠️ FAULTY (not working properly)

---

## 📂 Dataset

In this project, we generate a **simple simulated ELPV-like dataset** inside Google Colab itself.  
No external dataset download is required.
